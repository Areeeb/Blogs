Creating CSS Grid Layout Masonry

What is a masonry brick


Masonry is a JavaScript grid layout library. It works by placing elements in optimal position based on available vertical space, sort of like a mason fitting stones in a wall. You’ve probably seen it in use all over the Internet.

In today’s internet, a better site is considered to be the one that makes amount of data presented inversely proportional to the space and time that is consumed. People today are so habitual of looking at web pages that doesn’t look so old school and they have down to look at a complete data say it be an image or a paragraph or anything. The best way to render data is to show the maximum of it within the first glance maintaining the quality and clarity. You feel your users are tired of scrolling down to your bunch of images, make a grid layout and you are good to go.

But does making a grid layout solves your problem? What if you have images of multiple sizes and dimensions, then a grid layout will look something like this.

Not so impressive, right?

How about aligning it using masonry, so that it looks cleaner, aligned and impressive like this one.

The images have been added in same order in HTML in both cases, masonry automatically detects which images would align the images in a proper fashion, and lets your site render a nice clean and aligned grid of images in all devices with different aspect ratios.

How to implement?


It is preferred that jquery should be downloaded or at least referenced into your project as jquery is much faster. You can install jquery through node package manager by command:

npm install jquery

Or you can include link to jqeury source version online using

<script type="text/javascript" src="http://code.jquery.com/jquery-1.4.2.min.js"></script>

There are a whole lot of links with which you can use functions of jquery.


HTML:

You will need to make a reference to masonry package. You can download both minified or unminified versions of masonry package from their website

After that, you add reference to masonry script.

<script src="/your-directories/masonry.pkgd.js"></script>

or

<script src="/your-directories/masonry.pkgd.min.js"></script>

You can also make a link to their script online.

<script src="https://unpkg.com/masonry-layout@4/dist/masonry.pkgd.js" ></script>

Make a div and add “grid” class to it. Then insert your child divs or child images into it. Add “grid-item” class to your child tags.

<div class="grid"> 
    <img class="grid-item" src='../Your/Image/Path' alt="Image">
    <img class="grid-item" src='../Your/Image/Path' alt="Image">
    <img class="grid-item" src='../Your/Image/Path' alt="Image">
    <img class="grid-item" src='../Your/Image/Path' alt="Image">
    <img class="grid-item" src='../Your/Image/Path' alt="Image">
    <img class="grid-item" src='../Your/Image/Path' alt="Image">
    <img class="grid-item" src='../Your/Image/Path' alt="Image">
</div>

CSS:

Add styles to your “grid-item” class.

.grid-item {
    font-size: 30px;
    text-align: center;
    width: 200px;
}

JavaScript:

Use jquery selector to grab the element with class name “grid” which is the parent div of the masonry brick and use the masonry plugin to call masonry constructor and pass in options like item-selector and columnWidth into it. Remember to follow camel case as JavaScript follows camel Case practice.

$('.grid').masonry({
    // options
    itemSelector: '.grid-item',
    columnWidth: 200
});

Execute the code and you will see how your grid images rendered in a much responsive smooth manner.
